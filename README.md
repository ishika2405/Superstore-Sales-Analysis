#  Superstore Sales Analysis

## Overview

This project analyzes retail sales data from the Superstore dataset using Python, SQL, and Power BI. The objective is to identify sales trends, customer behavior, and profit opportunities through interactive dashboards.

## Business Problem

Retail companies need data-driven insights to improve sales, reduce losses, and understand customer purchasing patterns.

## Tools Used

* Python (Pandas, NumPy)
* SQL (MySQL)
* Power BI
* Excel

## Project Workflow

1. Data Cleaning using Python
2. Data Analysis using SQL
3. Dashboard Creation using Power BI
4. Business Insights and Recommendations

## Dashboard Features

* Total Sales
* Total Profit
* Total Orders
* Monthly Sales Trend
* Sales by Region
* Sales by Category
* Top Customers
* Top Products
* Interactive Filters

## Key Insights

* Technology generated the highest revenue.
* West region had the highest sales.
* Discounts negatively impacted profit for some products.
* A few customers contributed a large share of total revenue.
